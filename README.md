# dashboardItj

#### Trabalho de visualização dos dados do relacionados ao Coronavírus no município de Itajaí.
#### As fontes de dados são os boletins epidemiológicos divulgados pela própria prefeitura diariamente.
#### Repositório com os arquivos dos códigos em linguagem R para criação de um dashboard e um relatório em markdown além dos arquivos csv contendo os dados utilizados na análise.
